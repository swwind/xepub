xepub - a lightweight epub reader

USAGE

  $ xepub [options] [file]

  Alavilable options:

  -p, --port               http port (default 23333)
  -o, --open               open browser automaticly
  -h, --help               show help document and exit
  -v, --version            show xepub version and exit
      --debug              show debug messages
      --gencert            generate a certificate so as to enable https and wss
      --https              use https (requires a certificate)
      --http               use http

EXAMPLE

  $ xepub --gencert
  $ xepub erohon.epub
  $ xepub -o -6 erohon.epub
  $ xepub -p 4000 erohon.epub
